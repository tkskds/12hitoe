
## DirDescription

  - images

      filters in DynamicHeader, NotFound images, favicon, default thumbnail and so on...

  - lib

      original CSS/JS/functions(php) files

  - parts

      component files

  - vendor

      3rd party files(Materialize , FontAwesome, and so on)
